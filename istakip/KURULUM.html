İş Takip — Kurulum Rehberi

5 adımda ücretsiz yayına alın. Teknik bilgi gerekmez.

1
GitHub hesabı aç
github.com adresine git ve ücretsiz hesap oluştur.
Zaten hesabın varsa bu adımı atla.
2
Dosyaları GitHub'a yükle
GitHub'da New repository → Ad: istakip → Create.

Ardından uploading an existing file linkine tıkla ve sana verilen istakip klasörünün içindeki şu dosyaları sürükle bırak:

• package.json
• vercel.json
• api/data.js → önce api klasörü oluştur
• public/index.html → önce public klasörü oluştur

Commit changes butonuna bas.
💡 Klasör yapısı önemli — dosyaları doğru klasöre koy.
3
Vercel hesabı aç
vercel.com → Sign Up → Continue with GitHub ile giriş yap.
Bu sayede Vercel GitHub hesabını görür.
4
Projeyi Vercel'e bağla
Vercel dashboard'unda Add New → Project.
istakip reposunu bul → Import → Deploy.

Birkaç saniye bekle, Vercel otomatik olarak kurar.
5
Linki takımla paylaş
Deploy bitince Vercel sana bir link verir:
istakip-xxxxx.vercel.app

Bu linki takımdaki herkese gönder. Hepiniz aynı veriyi görür, değişiklikler 15 saniyede yansır.
💡 Linki yer imlerine ekle, her zaman bu adresten gir.
Hepsi bu kadar! Ücretsiz, reklamsız, sonsuza kadar çalışır.
